How to: Visualizations using D3.js

Draw a circle using D3

This section will start with a simple example illustrate on how to generate a circle with a radius provided by user.

Before we actually draw a circle, let’s talk about the D3 select API and append API. D3 select API uses same method as CSS selector. So .select("body") will select the element: the "body" tag. The selector will tell the web page where to insert the graphic we will generate. To insert the graphic, we will use the append API. After we’ve selected the body tag, now we could use .append("svg") to add a SVG(Scalable Vector Graphics) element into the body.

Let’s first create a fixed sized circle. The circle will have a radius of 50 pixels. And with the color “red”. Here is the code:

 					<svg width="100" height="100">
					<circle cx="50" cy="50" r="50" fill="red" />
					</svg>

In this example, “cx” and “cy” define the x-axis and y-axis coordinate of the center of the element. “r” defines the radius of the circle. And “fill” defines the color filling the circle. You will see a circle looks like this:

Now we want to use D3 API to insert this circle into “body”, we could use the chain syntax:

					d3.select("body").append("svg").attr("width", 50).attr("height", 50).append("circle").attr("cx", 25).attr("cy", 25).attr("r", 25).style("fill", "purple");

As we’ve talked above, the above code does these:

  1. Select “body” tag.
  2. Append sert a SVG element to the last of “body”.
  3. Set all the attritubes using “attr” and “style”.

But this code looks a little bit messy. Let’s make it clearer. D3 can set all these attributes to a viariable called “section”, now we can set the SVG to svgSection, and all the circle attributes to circleSection:

				var bodySelection = d3.select("body");
				var svgSelection = bodySelection.append("svg")
					.attr("width", 50)
					.attr("height", 50);
 				var circleSelection = svgSelection.append("circle")
 					.attr("cx", 25)
 					.attr("cy", 25)
					.attr("r", 25)
					.style("fill", "purple");

We can change the radius and the color of the circle by changing the content after “r” and “fill”.
