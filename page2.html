How to: Visualizations using D3.js

Draw basic shapes using D3

We have gone through how to use D3 to draw a circle. In this section, we will generate several basic shapes using D3. The basic steps to use D3 to draw a shape are:

  1. Make an SVG container.
  2. Draw the shape.

Rectangle

Let's start to draw a rectangle. The code of drawing a rectangle using D3 is: 

 					
 					//Make an SVG Container
 					var svgContainer = d3.select("body").append("svg")
 						.attr("width", 200)
 						.attr("height", 200);
 					
 					//Draw the Rectangle
 					var rectangle = svgContainer.append("rect")
 						.attr("x", 10)
 						.attr("y", 10)
 						.attr("width", 200)
 						.attr("height", 100);

The “x” and “y” attributes define the left and top positions of the rectangle. It is obvious that the “width” and “height” attributes are the width and height of the rectangle.

Ellipse

The code of drawing an ellipse using D3 is: 

            	
            			//Make an SVG Container
				var svgContainer = d3.select("body").append("svg")
					.attr("width", 200)
					.attr("height", 200);
				//Draw the Ellipse
				var circle = svgContainer.append("ellipse")
					.attr("cx", 100)
					.attr("cy", 100)
					.attr("rx", 100)
					.attr("ry", 50);

The “x” and “y” attributes define the coordinate of the center point of the ellipse. The "rx" attribute defines the horizontal radius. The "ry" attribute defines the vertical radius

Straight Line

The code of drawing a straight line using D3 is: 

            	
            			//Make an SVG Container
				var svgContainer = d3.select("body").append("svg")
					.attr("width", 200)
					.attr("height", 200);
				//Draw the Ellipse
				var circle = svgContainer.append("line")
					.attr("x1", 10)
					.attr("y1", 10)
					.attr("x2", 150)
					.attr("y2", 150);

The “x1” and “y1” attributes define the coordinate of start point of the straight line. The “x2” and “y2” attributes define the coordinate of end point of the straight line.
