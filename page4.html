How to: Visualizations using D3.js

Review

From this how-to guide, you should have learned how to use D3 APIs to draw circle, rectangle, straight lines. And use data sets to create a simple pie chart.

More specifically, we’ve used such APIs in our how-to guide:

  • D3.select - select an element from the document.
  • selection.append - create, append and select new elements.
  • selection.data – join elements to data.
  • d3.scaleOrdinal -create an ordinal scale.
  • d3.schemeCategory20b - a categorical scheme with 20 colors.
  • d3.arc - create a new arc generator.
  • d3.pie - create a new pie generator.

There are many things D3 could do, if you are interested, you could go to their website to learn more.
